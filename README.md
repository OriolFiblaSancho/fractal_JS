# Fractal Generator

Interactive fractal explorer built with HTML, CSS, and JavaScript. Zoom, pan, and switch between multiple fractals in the browser.

## Live demo
[Try it here →](https://orifibla.tech/fractal_JS/)

## Fractals included
- Koch Snowflake
- Sierpinski Triangle
- Mandelbrot

## Controls
- Buttons: switch fractal, zoom in/out
- Mouse: drag to pan; mouse-wheel zoom (if enabled)

## Warning
⚠️ High fractal depth or extreme zoom can cause lag or freeze the browser.

Made with plain JavaScript for fun and learning.
